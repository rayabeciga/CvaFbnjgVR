# SpringBoot仓库物流管理系统（源码+文档+PPT）简介

## 核心功能
- 用户登录
- 用户注册
- 商品管理
- 供应商管理
- 客户管理
- 进货管理
- 发货管理
- 交付管理
- 权限管理
- 角色管理
- 用户管理
- 部门管理
- 菜单设置

## 适用场景
- 仓库物流企业
- 物流服务公司
- 供应链管理公司

## 技术特点
- 基于 Java 和 Spring Boot 框架
- 使用 MySQL 数据库
- MyBatis-Plus 持久层框架
